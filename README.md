# Translate-WordPress-Slider-Revolution

WP Slider Revolution has WPML support but lack of native Polylang support. This code snippet is to fake that the WPML plugin is loaded and the slider revolution will enable all the translation functions and allow users to translate sliders with Polylang plugin.

Required Plugin: Polylang
https://wordpress.org/plugins/polylang/

Follow this guide to learn how to translate slider.
https://wpml.org/documentation/plugins-compatibility/creating-multilingual-sliders-with-revolution-slider-and-wpml/
